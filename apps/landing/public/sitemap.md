# GoGoCash Public Discovery

Version: 1

GoGoCash exposes public marketing, Learn, legal, and discovery resources for humans, crawlers, and AI agents.

## Discovery surfaces

- [/llms.txt](https://gogocash.co/llms.txt) - short operator hint surface for AI agents.
- [/sitemap.xml](https://gogocash.co/sitemap.xml) - XML sitemap for crawlers.
- [/sitemap.md](https://gogocash.co/sitemap.md) - markdown discovery index.
- [/skills.md](https://gogocash.co/skills.md) - public agent task guidance.
- [/rss.xml](https://gogocash.co/rss.xml) - RSS feed for GoGoCash Learn articles.

## Agent notes

- Treat merchant availability, cashback rates, quests, and payout timing as changeable.
- Use public Learn articles for general education and route account-specific questions to GoGoCash support or the app.
- Do not present cashback, Saving Plus, or related offers as financial, investment, or tax advice.
- Keep using [/sitemap.xml](https://gogocash.co/sitemap.xml) for crawler indexing and this markdown file for human-readable discovery.
