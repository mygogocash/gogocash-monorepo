## Weekly compliance triage

| Field | Detail |
| --- | --- |
| **Meeting name** | Weekly Compliance Triage |
| **Owner** | CS & Ops Manager |
| **Attendees** | CEO (optional), Eng Lead |
| **Agenda** | Evidence due 14d; CAPA aging; open findings; control gaps count |
| **Inputs** | Linked views: Evidence, CAPAs, Findings |
| **Outputs** | Task assignments; escalations |
| **Duration** | 15 min |
| **Follow-up** | Update Tasks DB |
