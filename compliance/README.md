# GoGoCash — Compliance & Integrated Management System

This directory contains the **lean IMS** for **SOC 2 Type II** (Security, Availability, Confidentiality, Processing Integrity), **ISO 9001**, and **ISO 27001:2022**.

## Start here

1. `ASSUMPTIONS_AND_SCOPE.md` — baseline and boundaries  
2. `01-scope/SCOPE-001-scope-statement.md` — formal scope  
3. `04-controls/INTEGRATED-CONTROL-MATRIX.md` — master controls  
4. `../CONTROLLED_DOCUMENT_INDEX.md` — approved document list

## Artifact types

Each document states whether it is **policy**, **procedure**, **record**, **template**, **technical control**, or **evidence**.

## Approval

Policies require **CEO** approval before relying on them for audits. Track versions in git and in `CONTROLLED_DOCUMENT_INDEX.md`.
